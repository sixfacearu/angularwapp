Step to Create Basket

1.Select Tokens: Select the Tokens that you currently have for which you want to create a Token Basket.

2. Basket creation: A smart contract will be created that will hold the underlying Tokens.

3. Authorization: Authorize the smart contract to enable hold and transfer of the underlying Tokens.

4. Deposit: Deposit the underlying Tokens from your wallet into the Basket smart contract.

5. Publish: Publish the Token Basket onto the Ethereum Blockchain.

*Every Token Basket is a smart contract. Ownership of the smart contract is transferred when a trade occurs. This is entirely decentralized, and WandX holds no code but the front-end to enable this creation and trade of Token Baskets.
